# Date Roulette ❤️

Never wonder **"What should we do today?"** again.

**Date Roulette** is a fun and interactive web app that helps couples discover personalized date ideas with a single spin. Instead of endlessly deciding where to go, users simply choose their preferences and let the wheel create a unique date experience.

## 🚀 Try It Live

**Live Website:** https://date-roulette-topaz.vercel.app/

## ✨ Features

* 🎡 Spin the wheel for a unique date every time
* 💰 Personalized suggestions based on your selected budget
* 🌦️ Date ideas that adapt to the weather
* 📍 Venue-based recommendations (Stay Home, Nearby, Far Away, Different Country)
* 🔄 Different date ideas on every spin with minimal repetition
* 🍽️ Curated food suggestions
* 💬 Fun conversation starters
* 🎵 Mood-based playlist recommendations
* 📸 Mini challenges to make every date memorable
* 🎁 Bonus surprise missions for extra fun

## 🚀 How It Works

1. Select your preferred **budget**.
2. Choose the current **weather/climate**.
3. Pick where you'd like your date to be:

   * 🏠 Stay Home
   * 🚶 Nearby
   * 🚗 Far Away
   * ✈️ Different Country
4. Click **Spin**.
5. Receive a personalized date plan

Every spin generates a fresh experience based on your selected preferences.
